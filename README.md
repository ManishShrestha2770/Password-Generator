# 🔐 Password Generator (Python)

A simple and beginner-friendly password generator built using Python.  
It allows users to generate secure passwords based on custom rules like length, numbers, and special characters.

---

## 🚀 Features

- Generate random secure passwords
- Custom password length input
- Option to include numbers
- Option to include special characters
- Ensures password meets selected requirements
- Beginner-friendly Python logic

---

## 🛠️ Technologies Used

- Python 3
- `random` module
- `string` module
